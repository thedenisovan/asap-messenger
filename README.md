## ASAP Messenger

### Description
A full-stack messaging web application built to demonstrate modern frontend and backend development skills.

The app is fully responsive, follows WCAG accessibility guidelines, and supports real-time communication across devices.

### Preview
You can wisit my website by clicking [this link](https://asap-messenger.pages.dev/). 

### Features
- Responsive design (mobile-first)
- Dark / Light mode
- User authentication
- Edit user profile
- Add / remove contacts
- Direct messaging
- Group chat creation and management
- Send messages in group chats
- Last online status tracking
- Clear chat history
- View group chat details

### Tech Stack

#### Full Stack
- TypeScript
- Socket.IO

#### Frontend
- React
- Tailwind CSS
- Vite
- React Testing Library
- Vitest

#### Backend
- Node.js
- Express
- PostgreSQL
- Prisma

#### Backend Testing & Utilities
- Jest
- Supertest
- CORS
- JSON Web Token (JWT)
- Express Validator
- bcryptjs
